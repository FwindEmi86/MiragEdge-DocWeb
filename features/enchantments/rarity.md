# 附魔品质系统配置说明
本服务器附魔系统采用多品质分级机制，不同品质对应专属配色及获取难度（权重），以下是全品质的详细配置（**权重数值越小，该品质稀有度越高、获取概率越低**）：

| 品质名称 | 配色（色值） | 权重（稀有度） |
|----------|--------------|----------------|
| 普通     | 汉白玉（#f8f4ed） | 1000（最易获取） |
| 罕见     | 毛绿（#66c18c） | 500 |
| 精良     | 霁青（#63bbd0） | 250 |
| 诅咒     | 鹤顶红（#d42517） | 25 |
| 史诗     | 夹竹桃红（#eb507e） | 50 |
| 传奇     | 淡橘橙（#fba414） | 10 |
| 稀世     | 油菜花黄（#fbda41） | 5 |
| 皮肤     | 粉团花红（#ec9bad） | 1（最稀有） |

### 补充说明
权重参数用于判定附魔品质的抽取概率，权重数值越大，该品质被抽取到的概率越高；反之则稀有度更高、获取难度更大。
