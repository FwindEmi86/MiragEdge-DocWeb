# 锐界幻境 - 玩家公会系统
本服务器公会系统围绕团队协作、专属福利、竞技互动设计，覆盖管理、成长、玩法等多维度，以下是核心功能的详细说明：

## 📌 基础管理功能
| 功能名称       | 功能详情                                                                 |
|----------------|--------------------------------------------------------------------------|
| 公会主城       | 会长可自定义设置游戏内任意位置作为公会主城，作为成员核心聚集点             |
| 公会传送       | 公会成员可通过专属功能一键传送至公会主城，无需手动跑图                   |
| 公会成员管理   | 公会管理员可在GUI操作界面中，便捷完成成员添加/移除、权限调整、职位设置等操作 |
| 公会邀请       | 管理员/会长可主动向在线玩家发送入会邀请，简化招新流程                     |

## 🎁 福利与成长体系
| 功能名称       | 功能详情                                                                 |
|----------------|--------------------------------------------------------------------------|
| 公会签到       | 成员每日完成签到可领取对应奖励，奖励与公会等级/活跃度挂钩                 |
| 公会银行       | 成员向公会贡献的物资/货币，将统一转化为公会资金，用于公会升级、功能解锁等 |
| 公会称号       | 称号随公会等级提升解锁，支持自定义称号名称；解锁后附带专属Buff效果        |
| 公会神石       | 按服务器配置的周期自动为公会增加活跃度，助力公会等级提升                 |

## 🌍 专属场景功能
### 公会世界
若开启该功能，会长可消耗星痕石（点卷）、灵叶（金币）或公会资金，创建带有边境限制的公会专属世界，供公会成员专属使用（如集体建造、资源采集等）。

## ⚔️ 竞技互动玩法
| 功能名称       | 功能详情                                                                 |
|----------------|--------------------------------------------------------------------------|
| 公会活动       | 目前开放两类公会战：<br>① 赛季公会战（周期性赛季制对抗）<br>② 匹配公会战（实时匹配其他公会对抗） |
| 公平对决       | 可开启“公平匹配/公平对决”模式，对战时平衡双方基础属性，保障竞技公平性     |

## 🛒 交易流通功能
### 公会商店
公会成员可在专属商店内自主上架物品出售，供其他公会成员选购，实现公会内部物资流通与共享。
